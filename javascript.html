자바스크립트가 동적이라는 말이 위에서는 크게 와닿지 않았다.
개인적으론 이벤트를 공부하면서 자바스크립트가 동적이라는 것에 대해 조금은 더 알 수 있었던 것 같다.
아래와 같이 버튼을 하나 생성해보자.



코드는 이러하다.

<input type="button" value="hi" onclick="alert('hi')">

  • input 태그로 타입이 버튼이라는 것을 생성한다
  • 버튼에 들어갈 텍스트를 넣어주려면, value 라는 속성에 값을 넣어준다
  • 버튼을 누르면 수행할 동작은 onclick=""이라는 속성을 사용한다
  • alert()은 웹브라우저에 경고창을 띄워주는 자바스크립트 함수이다
  • alert()의 () 안에 출력할 문구를 쓰면 된다
  • 문자열은 '' 안에 'hi' 이런 식으로
  • 숫자는 그냥 1234 이렇게 쓰면 된다
여기까지 작성했다면 버튼을 눌러서 어떤 일이 발생하는지 확인해보자.
아래는 text 타입

<input type="text" onchange="alert('changed')">



아래는 onkeydown 속성. 효과는 직접 테스트해보자.

<input type="text" onkeydown="alert('changed')">



위와 같이 on으로 시작하는 속성을 이벤트라고 한다.

제어할 태그 선택하기



input 태그에 타입을 버튼으로 하면 페이지에 버튼이 생성된다.

<input type="button">


아무것도 없는 버튼이 만들어지는데, 여기에 글자를 넣으려면 value 속성을, 버튼 클릭에 대한 반응을 넣으려면 onclick 속성을 추가해준다.

<input type="button" value="night" onclick="">

글자가 추가되었지만 버튼을 눌러도 여전히 특별한 동작은 없다.
이제 onclick=""속성에 JavaScript 로 원하는 동작의 코드를 작성해주면 된다.
야간모드를 만들기 위해 배경을 검정으로, 텍스트를 흰색으로 변경해보자.
아래와 같이 코드를 써준다.

<input type="button" value="night" onclick=" document.querySelector('body').style.backgroundColor='black'; document.querySelector('body').style.color='white'; ">

JavaScript 코드를 쓰려면 document 를 쓰고, 동작을 적용할 태그를 반영 시켜준다.

document.querySelector('body')

그리고 해당 태그의 스타일의 백그라운드 컬러를 검정으로 적용 시켜준다. (끝에는 세미콜론)

document.querySelector('body').style.backgroundColor='black';

같은 방식으로, 텍스트의 컬러는 흰색으로 적용 시킨다.

document.querySelector('body').style.color='white';

적용해보면 아래와 같은 버튼이 생성되고, 버튼을 누르면 다크 모드가 발동(!!)된다.
다크모드만 있으면 심심하니까, 일반모드 버튼도 같이 만들어주자.
앞의 코드에서 value를 day로, 색상을 white, black 으로 바꿔주면 된다.

<input type="button" value="day" onclick=" document.querySelector('body').style.backgroundColor='white'; document.querySelector('body').style.color='black'; ">

  • ===
    • 좌우의 값이 같은지를 검사 후 같으면 true, 다르면 false 가 출력된다.
    document.write(1===1) / document.write(1===2) 로 확인해보자.

  • <

  • >
    • 위 비교연산자와 반대다.
    왼쪽의 값이 오른쪽보다 큰지를 검사한다.

function setColor(color){

i++;
}
}

function setColor(color) 함수의 매개변수로 color 값을 전달해 텍스트의 색상을 변경할 수 있다.

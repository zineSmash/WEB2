React 실습환경 구축

https://opentutorials.org/course/4900/31263
  1. react 홈페이지 방문
  2. 시작하기 버튼 클릭
  3. 우측 메뉴에서 새로운 React 앱 만들기 클릭
  4. 추천 툴체인에서 Create React App 클릭
  5. 깃허브 페이지가 뜨면, 우측의 create-react-app.dev 를 클릭
  6. 여기서 잠시 멈추고, 다른 페이지를 열어서 nodejs.org 사이트로 이동
  7. 가장 최신 버전의 node.js 를 설치해준다. (나는 LTS 버전을 설치했다)
  8. node.js 설치가 끝났다면, VScode 를 실행
  9. react 로 작업할 폴더를 설정
  10. VScode 의 메뉴에서 터미널-새터미널 선택
  11. VScode 의 하단에 터미널이 실행되면 아래의 명령어 입력
  12. npx create-react-app . (콤마까지 찍어준다)
  13. 중간에 무언가 인스톨하라는 메세지가 나오면 y 를 눌러 인스톨해준다
  14. 위 작업이 끝나면, npm start 라는 명령어를 입력
  15. react 개발환경이 실행되고, 웹브라우저 창이 켜지며 샘플 react 앱이 실행된다
  16. 박수!!
